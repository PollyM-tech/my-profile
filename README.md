# Personal Website Profile

Welcome to my personal website project! This is a simple website that showcases my profile as a software engineer. It includes a homepage, a profile picture, and links to my social media accounts. The website is **responsive**, meaning it works well on both desktop and mobile devices.

This project is a great way to learn the basics of **HTML**, **CSS**, and **JavaScript**. Let’s dive in!

## What’s in This Project?

This project is a simple personal website with the following features:
- A **header** with navigation links (Home, About, Goals, Contact).
- A **main section** with my name, a short description, and a "Say Hello" button.
- A **profile picture**.
- A **footer** with links to my social media accounts (GitHub, LinkedIn, Email).
- **Responsive design**: The website looks good on both desktop and mobile devices.
- **Interactive features**: A button that shows a greeting message when clicked.

## What Each File Does

### 1. **`index.html`**
This is the main file for the website. It defines the structure of the website using **HTML**. Here’s what it includes:
- A **header** with navigation links.
- A **main section** with my name, a short description, and a "Say Hello" button.
- A **profile picture**.
- A **footer** with social media links.

### 2. **`css/style.css`**
This file styles the website using **CSS**. It includes:
- Colors, fonts, and layout for the website.
- **Media queries** to make the website responsive (look good on mobile devices).

### 3. **`src/index.js`**
This file adds interactivity to the website using **JavaScript**. It includes:
- A function to change the background color based on the screen size.
- A "Say Hello" button that shows a greeting message when clicked.
